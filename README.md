# Calendar-web

Веб-приложение для управления календарем событий с использованием современных технологий React и Redux Toolkit. Проект позволяет пользователям создавать, редактировать и просматривать события в удобном календарном интерфейсе.

## Установка

1. Клонируйте репозиторий:
   ```bash
   git clone <url-репозитория>
   ```

2. Перейдите в директорию проекта:
   ```bash
   cd calendar-web
   ```

3. Установите зависимости:
   ```bash
   npm install
   ```

4. Запустите приложение в режиме разработки:
   ```bash
   npm run dev
   ```

5. Для запуска тестов:
   ```bash
   npm run test
   ```

Основные технологии: React 19, Redux Toolkit, FullCalendar, VKUI, TypeScript, Vite, Jest
